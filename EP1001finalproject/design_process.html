Design Process

Before I start designing anything, I always do research and write out all limitations and issue I might face.
Note down the list of requirements the item is required to help with the skatching and prototyping process.

  1. List down requirements, expectation, limitations of the item
  2. Research on item done by others or your past projects.
  3. Sketches and protyping(My favorate part)
  4. Test fit, final modifications, data collection and Documentation
    (Repeat step 3 if you have to)
  5. Item completed

Project requirements

This are the things needed to be in your product

  1. 2D and/or 3D designed part(s)
  2. Subtractive manufacturing (i.e. cnc) part
  3. An embedded microntroller that I designed & fabricated)
  4. Input and/or output device(s) that interacts with the user
  5. Embedded program
  6. Documated on website
  7. Electronics should be neatly mounted and wiring neatly laid out and not simply just thrown together in the case ← LMAO
